plasma-activities-stats-6.4.4

Introduction to plasma-activities-stats-6.4.4

The plasma-activities-stats-6.4.4 library provides access to the usage data collected by the KDE Activities system. It is normally built with Plasma-6.4.4 but is included here because it is needed for kio-extras-25.08.0.

This package is known to build and work properly using an LFS 12.4 platform.

Package Information

  • Download (HTTP): https://download.kde.org/stable/plasma/6.4.4/plasma-activities-stats-6.4.4.tar.xz

  • Download MD5 sum: e19f331b7233dfa00c1f9a4dc282aa6d

  • Download size: 84 KB

  • Estimated disk space required: 4.0 MB

  • Estimated build time: 0.2 SBU

plasma-activities Dependencies

Required

plasma-activities-6.4.4

Installation of plasma-activities-stats

Install plasma-activities-stats by running the following commands: 

mkdir build &&
cd    build &&

cmake -D CMAKE_INSTALL_PREFIX=$KF6_PREFIX \
      -D CMAKE_BUILD_TYPE=Release         \
      -D BUILD_TESTING=OFF                \
      -W no-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
