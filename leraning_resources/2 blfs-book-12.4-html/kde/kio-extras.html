kio-extras-25.08.0

Introduction to kio-extras

The kio-extras package contains additional components to increase the functionality of the KDE resource and network access abstractions.

This package is known to build and work properly using an LFS 12.4 platform.

Package Information

  • Download (HTTP): https://download.kde.org/stable/release-service/25.08.0/src/kio-extras-25.08.0.tar.xz

  • Download MD5 sum: 1d11770bdb9cb4549814c98a10f43f26

  • Download size: 3.4 MB

  • Estimated disk space required: 93 MB

  • Estimated build time: 0.6 SBU (using parallelism=4)

kio-extras Dependencies

Required

KDE Frameworks-6.17.0, kdsoap-ws-discovery-client-0.4.0, libproxy-0.5.10, plasma-activities-stats-6.4.4, and qcoro-0.12.0

Recommended

libkexiv2-25.08.0

Optional

libtirpc-1.3.6, Samba-4.22.4, taglib-2.1.1, libappimage, libimobiledevice, libmtp, libplist, libssh, and OpenEXR

Installation of kio-extras

Install kio-extras by running the following commands: 

mkdir build &&
cd    build &&

cmake -D CMAKE_INSTALL_PREFIX=$KF6_PREFIX \
      -D CMAKE_BUILD_TYPE=Release         \
      -D BUILD_TESTING=OFF                \
      -W no-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
